## Goal
+ C 언어 이해
+ Github or Space(협업툴) 사용법 이해
+ 자신이 작성한 코드 분석
## Basic
+ ~3.17 까지 모집
+ 3.18 일정 공유 및 멘토링 날짜 지정
+ 주 1회 30~1시간 가량 강의 및 과제 설명
+ 과제는 다음 강의 1일 전까지 제출 e.g. 다음 강의가 3.21이면 3.20까지 제출
+ 구글 미팅 + 카톡 and Github project 이용
+ 사전 준비: 희망 시간 or 시간표 조사. Github 아이디 만들기. 백준 아이디 만들기
